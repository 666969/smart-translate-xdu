# 智译西电比赛演示脚本

## 1. 赛前 2 分钟检查清单

- 确认 `.env.local` 已配置 `OPENAI_API_KEY`、`OPENAI_MODEL`、`OPENAI_MODEL_FLASH`、Firebase 公共变量。
- 用 `npm run dev` 或已部署地址提前打开项目，确认首页、`/pdf`、`/live`、`/notebook` 能正常进入。
- 浏览器优先使用最新版 Chrome；如果要演示 `/live` 标签页音频，提前确认浏览器支持“共享标签页并勾选共享音频”。
- 如果要演示登录，提前在 Firebase Console 中启用 Email/Password 和 Google Provider。
- 准备 1 份可提取文本的 PDF；另外准备 1 份扫描版 PDF 作为备选。
- 首页片段速译已经有“填入演示文本”按钮，现场没有素材时可以直接使用。
- 如果现场网络一般，优先演示首页和 `/notebook`，把 `/live` 放到后面作为加分项。

## 2. 推荐 3-5 分钟演示路径

1. 首页 `/`
一句话亮点：把外文教材片段直接转成中文译文、解析和专业术语。

2. 首页 Document 模式
一句话亮点：多张课件截图自动生成知识脉络和三语小测，答错自动沉淀到错题本。

3. `/pdf`
一句话亮点：上传完整 PDF 后，自动生成摘要，并支持围绕页码上下文继续追问。

4. `/live`
一句话亮点：课堂音频或麦克风内容可实时转成双语字幕，并导出同传记录。

5. `/notebook`
一句话亮点：把前面演示产生的生词和错题统一汇总成复习资产。

## 3. 每个页面建议讲词

### 首页 `/`

- 讲词：这个页面解决的是“外文教材看得慢、看不透、术语记不住”的问题。
- 操作：点击“填入演示文本”，再点击“开始速译”。
- 预期结果：右侧出现中文译文、深度解析、关键词卡片。
- 补一句：这里的关键词可以直接收藏到生词本，结果也能一键导出 Markdown 笔记。

### 首页 Document 模式

- 讲词：如果学生面对的不是一段文字，而是一组课件截图，就切到“课件精读”。
- 操作：上传 1 到 5 张课件图，点击“开始精读解析”。
- 预期结果：页面生成 Mermaid 知识脉络和 3 道三语小测。
- 补一句：如果我故意答错一道题，它会自动进入错题本，后面在 `/notebook` 里继续复习。

### `/pdf`

- 讲词：这个页面面向完整文献或讲义，不再是碎片，而是整份 PDF。
- 操作：上传 PDF，等待右侧摘要生成，再提一个具体问题。
- 预期结果：右侧先出现全文摘要，再出现 AI 助教回答；如果命中页码，会保留参考页码信息。
- 补一句：现在还支持导出文献精读笔记，比赛结束后可以直接留存。

### `/live`

- 讲词：这个页面模拟真实课堂场景，老师讲、学生听、系统同步出双语字幕。
- 操作：先说明可以选“标签页音频”或“麦克风”，然后点击“开始同传”。
- 预期结果：左侧持续出现原文和中文，右侧出现术语洞察卡片。
- 补一句：演示过程中我不会中断收音，当前快照也可以直接导出为同传记录。

### `/notebook`

- 讲词：前面所有交互不是一次性展示，而是会沉淀成长期学习资产。
- 操作：打开生词本、错题本，再点击“生成考前冲刺包”。
- 预期结果：看到收藏术语、错题记录和冲刺包摘要。
- 补一句：这意味着系统不是只会“翻译”，而是能形成学习闭环。

## 4. 现场故障兜底

### OpenAI 响应较慢

- 替代说法：现场模型返回稍慢，我先展示已经生成好的学习闭环和导出结果。
- 替代路径：优先切到 `/notebook` 或直接展示刚才导出的 Markdown。

### 标签页没有音频

- 替代说法：有些机器没有勾选“共享音频”，我改用麦克风口述，逻辑完全一致。
- 替代路径：切到 `/live` 的“麦克风”模式，读两句法语或英语样例。

### Firebase 未配置

- 替代说法：登录是增强能力，不影响核心翻译、精读和文献问答演示。
- 替代路径：跳过登录，直接展示首页、`/pdf`、`/live`、`/notebook`。

### PDF 被识别为扫描版

- 替代说法：这类 PDF 没有可靠文本层，所以系统自动切到按页视觉解析模式。
- 替代路径：在右侧输入页码或页码范围，例如 `3-5`，继续提问即可。

### 网络不稳定

- 替代说法：我先展示本地可见的结果沉淀与导出，强调这是比赛环境下的兜底能力。
- 替代路径：优先展示首页已生成结果、`/notebook`、`DEMO.md` 中的演示脚本和已导出的 Markdown 文件。

## 5. 最短保底演示路径（90 秒）

1. 打开首页 `/`，点击“填入演示文本”并开始速译。
2. 展示翻译结果、深度解析、关键词收藏。
3. 切到 Document 模式，展示知识脉络和小测入口。
4. 打开 `/notebook`，说明生词和错题已经形成学习闭环。
5. 如果还有时间，再切到 `/pdf` 或 `/live` 展示导出按钮作为加分项。
